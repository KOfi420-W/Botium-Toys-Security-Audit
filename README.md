## 🎯 Audit Scope  
The purpose of this audit was to evaluate the security posture of **Botium Toys**, focusing on:

- Organizational security requirements  
- Data protection practices  
- Network and system vulnerabilities  
- Compliance alignment  
- Risk exposure by asset type  
- Existing controls and their effectiveness  

The audit is structured to reflect **NIST Cybersecurity Framework (CSF)** domain alignment.
